# TestRepo

This repository contains JavaScript learning examples organized by chapter and topic.

## Latest Files Added

- `05_chapter_operators/21_String_Op.js` — string operations, concatenation, length, character access, and `console.log` behavior.
- `05_chapter_operators/22_Ternary_Op.js` — ternary operator example that checks if a numeric `age` value is considered adult.
- `05_chapter_operators/23_IQ.js` — demonstrates strict equality (`===`) vs. loose equality (`==`) operators with status code type checking.

## Other Operator Examples

- `05_chapter_operators/13_DataTypes.js` — data types basics.
- `05_chapter_operators/14_Assignment_Operator.js` — assignment operator usage.
- `05_chapter_operators/15_Arithmetic_Operator.js` — arithmetic operator examples.
- `05_chapter_operators/16_Comparsion_Operator.js` — comparison operator examples.
- `05_chapter_operators/17_Logical_Operator.js` — logical operator example.

## Notes

These scripts are meant to demonstrate core JavaScript concepts in small, standalone examples. New code has been committed and pushed to `origin/main`.
